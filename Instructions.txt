You will need to install Node.js for windows which is our package manager for typescript
Make sure to restart studio whenever you install a package.

If you use Node.js to install make sure that is allowed
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser

Install/Update:
npm install typescript --global  

Initialize:
tsc -init   

Build:
tsc --build

Clean:
tsc --build --clean

Run the program with Live the Server Extension!

For some reason ghost errors generate when you rename files in TypeScript, just restart studio when your done renaming.
